# Changelog

## v0.2.0

- Added payload version chips that refresh from upstream releases when the app opens.
- Added a manual refresh control for payload versions.
- Added IPv4 validation before sending payloads.
- Added a connection test button for configured payload ports.
- Added drag-and-drop support for custom payload files.
- Renamed the etaHEN card while keeping the fixed `2.6B` version chip.
- Updated the README and project presentation for public use.

## v0.1.0

- Initial public release.
- Native macOS SwiftUI app for TCP payload delivery.
- Built-in payload catalog and custom payload support.
- Downloadable DMG release.
